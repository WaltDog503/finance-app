# Financial Suite - Gold Edition

A comprehensive personal finance application built with Python and Tkinter.

## Features
* **Simple & Compound Interest:** Calculate growth over time.
* **FV / PV Calculator:** Investment analysis tools.
* **Loan Amortization:** Monthly payment calculator with full schedule generation.
* **Budget Manager:** Track income/expenses and export to CSV.

## How to Run
1. Install Python 3 and Tkinter.
2. Run `python3 finance_app.py`.
